AIS Decoder
The decoder will decode all the NMEA and AIS content of all 27 AIS message types (!AIVDM !AIVDO etc) including many binary messages.
Input:      Serial or USB from AIS receiver, UDP or TCP from network, Log File
In Filter:   Message type, DAC, FI, ID, MMSI
Output:    Display, File, FTP, UDP
              As received or at scheduled intervals
Out Filter:Lat/Lon or any other decoded field
Format:   Text, CSV, HTML, XML, KML, KMZ, NMEA, Token Substitution
